
## EchoClient.java EchoServer.java

运行多个 EchoClient, 客户端访问服务端 EchoServer

1. EchoClient 客户端发送数据给服务端
2. 服务端将收到的 EchoClient 的数据转发给其他的 EchoClient

这样一个简单的群聊工具就可以用了, 不过还有很多问题, 后面慢慢完善.
